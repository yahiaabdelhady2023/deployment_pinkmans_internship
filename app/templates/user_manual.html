Homepage

Homepage: This page provides a breakdown of items and categories through historical graphs. You can view data for one branch or all branches. By default, the first graph displayed shows all items across all venues for all the period

By Default Select Type of Report is set to Specific Item Graph [Unit Sold], and that allows user to select a specific Item/All items for graph displaying. Note it shows unit not net sale

Graphs

The Graphs Page focuses on data aggregation and displays branch performance, along with the Labour Total Cost as a percentage. You can choose the time period for the charts, and Pie Chart, Bar Chart, or Doughnut Chart are the recommended options. There are three key metrics to view: Net Revenue (which shows the Labour Total Cost as a percentage along with the net revenue), Top Sold Items, and Top Sold Categories. To use the feature, simply select the desired time period (start and end dates), then choose the branch or select 'All' to include all branches. Additionally, you can view the detailed table by clicking the zoom-in button

Machine Learning

The forecast for next week is provided per item and branch. The 'Forecast' represents the predicted sales value for the upcoming week under typical conditions. The 'Upper Forecast' indicates the best-case scenario, reflecting the maximum number of items that could be sold. In contrast, the 'Lower Forecast' represents the worst-case scenario, showing the minimum sales expected for each item at each branch.

Settings

To zoom in (recommended for phone/tablet users), click on the red arrow next to the navbar. This will hide the navbar, providing more space for viewing the graphs
